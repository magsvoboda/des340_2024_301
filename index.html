One pager design

Subtitle

In our first exercise, we will design a so called one pager site. One pagers are websites from a single page.
Most of them has a single purpose: to make people to do something. It is Call to action.

How to do that?

Select a project from kickstarter.com. The purpose of the website is to make people to back the project.

Here are some guidelines:

  1. Select a real project, not a dream. This is a dream: Click here This is a project:
  2. Go to the master branch Pulsar Cirrus CI page.
  3. Select the latest successful build (check for a finished green check mark on the right side of the page - make sure you do not select a failed one).
  4. Select your system from the list of options: arm-linux - ARM based Linux systems (Raspberry Pi etc.) silicon_mac - M1 and M2 chip Apple Macs
  5. Select binary > binaries from the Artifacts pane
